# mindheaven-2026
## Author- Alisha bajaj 
MindHeaven is an AI journaling app bridging daily reflection and mental well-being. Built with Next.js, Supabase, and Hugging Face APIs.
<br>
Features:<br>
AI Mood Analysis: Tracks emotion trends via NLP.<br>
Privacy First: Ironclad Supabase Row Level Security (RLS).<br>
Dashboards: Visualizes mood patterns over time.
